# 時渡り草子 — 「夢間試験」設計書 v1.0

> 2026-07-12。指示：
> ①年表などを**参考資料**として見られるように（いまは学習内容が散っている）。世界観は崩さない。
> ②**確定した設定**＝令和の学校で歴史のテストを受けていたJKが寝落ちし、夢のなかでタイムトラベルしながら苦手な歴史を勉強する。
> ③この設定に照らして、ゲーム性・ストーリー・場面で足らない部分を洗い出して書き出す。中途半端な小改修では足りない。
> ④ゲーム冒頭に「テスト→寝落ち→時渡り」のアニメーションを置き、没入させる。

これにより spec.md §16 の未確定項目「タイムトラベルの動機」は**確定**：
**歴史の中間試験のさなかの寝落ち。夢が、苦手な時代へ連れていく。**

---

## 1. 設定の再定義（フレーム構造）

```
【現実・令和】 教室。歴史の中間試験。答案は空欄だらけ。
      │  寝落ち（まぶたが落ちる）
      ▼
【夢のあわい】 時の本棚（ホーム画面）。試験範囲の時代の巻が並ぶ。
      │  巻をひらく（篇を選ぶ）
      ▼
【夢のなか・過去】 その時代を数日暮らす（既存のゲーム本体）。
      │  最終日の眠り＝夢の底からの浮上
      ▼
【現実・令和】 目覚め。答案に、こたえが書ける。——体で覚えたから。
```

- 手に持っている**草子**の正体＝夢のなかで彼女の「勉強ノート」が化けたもの。
  見たこと・学んだことをひとりでに記録する。絵日記・年表・栞の器はこれで一本化される。
  （既存の作中セリフ「その古い草子、だいじそうに抱えて」等はこの解釈でそのまま生きる。書き換え不要。）
- 現実パートの美術＝**現代（明朝でなくゴシック、答案用紙の白、黒板の闇）**。
  夢パートの美術＝**和（和紙・墨・朱・金泥、縦書き明朝）**。
  「寝落ちの瞬間に書体と紙が変わる」ことが時渡りの視覚言語。世界観はこの対比で守る。

## 2. 足らない部分の洗い出し（書き出し）

### ストーリー
- **S1. 導入がない。** 旧設定（祖母の蔵の草子）は文字スライドのみで、テスト・教室・寝落ちが存在しない。→ **冒頭アニメーションで全面差し替え**（§3-A）。
- **S2. 目的が示されない。** 「なぜこの時代にいるのか」に答えがなかった。→ 目的＝**空欄だらけの答案を埋められるようになること**。冒頭で答案（空欄）を見せ、終幕で埋める。
- **S3. 学びが現実に還らない。** 終幕が「蔵で目覚めた」だけで、学んだことが役に立つ場面がない。→ **目覚めの答案シーン**：夢で見た出来事のこたえが、答案にひとりでに埋まっていく（§3-B）。埋まらなかった問いは責めずに「もういちど同じ夢を見れば」と再訪へ誘う。
- **S4. 篇と篇のつながりが弱い。** → 試験の**大問**構造で束ねる（大問一＝平安、大問二＝鎌倉、以後の篇は「次の試験範囲」）。平安の終幕で「大問二」を予告する。
- **S5. 主人公の声がない。** 夢のなかの彼女は時代人として扱われるだけで、「令和の受験生」の内面が漏れる瞬間がない。→ 名所で頁がひらいた際、**「——あ、これ、テストに出てたやつだ。」**という独白を一行だけ挟む（出題と重なった時のみ。押し付けの解説にはしない）。

### ゲーム性
- **G1. 学習の背骨がない（＝散っている）。** 年表・栞・図譜・宵の問いがバラバラの置き場にあり、「何を学べば何が起きるか」が繋がっていない。→ **答案（試験）タブ**を草子に挟む：出題一覧が常に見え、名所や暮らしで学ぶたびにこたえが埋まる。**学び＝答案が埋まる**という一本の糸で全部を束ねる（§3-C）。
- **G2. 年表が参考資料になっていない。** 未見の出来事は題名まで隠れており、「調べる」用途に使えない。→ **年表を全文公開の参考資料に**（年・元号・出来事・人物・一行・深掘りまで常に読める）。見た出来事には**朱印「見た」**が捺され、未見には「ゆかりの名所」への誘い文を添える。**収集の緊張は答案側に移す**（こたえは体験でしか埋まらない）。
- **G3. 日本史全体のどこにいるかが分からない。** → 年表の頭に**時代のながれ（縄文〜明治の帯）**を置き、いまの篇の位置を示す。ホームの本棚と同じ並びで、参考資料としても機能させる。
- **G4. 埋まった実感のフィードバック不足。** → 名所の語りの結びの独白（S5）と、試験タブの「n / m」表示。点数・成績は出さない（北極星：ゲージを見せない。**埋まる悦びだけ**にする）。

### 場面（欠けているシーン）
- **P1. 教室・試験のシーン**（冒頭アニメ）：チャイム→答案→解けない→まぶた→寝落ち→金泥の時渡り。**自動再生・スキップ可**。
- **P2. 目覚めのシーン**（終幕アニメ）：金の光→まぶたがひらく→教室→答案にこたえが走り書きで埋まる→独白→（平安なら）次の大問の予告。
- **P3. 音**：学校チャイム・時計の秒針・机に伏す音。夢側の虫の音（既存）と対になる**現実側の音**が無かった。WebAudioで合成（アセット不要・容量ゼロ）。
- **P4. 席次表的な小物**：答案用紙に「二年Ｃ組　名前」欄、試験範囲の記載。没入は小物が作る。

### 見送り（今回やらない・理由つき）
- 3D教室シーン：現実パートはあえて**平面（紙）**で描く。夢＝立体／現実＝紙という対比が世界観に合い、実装も堅い。
- 採点・点数表示：北極星に反する。埋まる／まだ、だけ。
- 宵の問い（夜の一問）の試験問題化：夜の問いは「今日見たものの復習」のままにする。試験は試験タブに集約。
- 教師・級友キャラ：冒頭アニメの匿名の気配（チャイム・紙の音）に留める。人物を増やすと現実パートが重くなる。

## 3. 実装設計

### A. 冒頭アニメーション `ClassroomIntro`（mode: prologue を差し替え）
DOM＋CSSキーフレームの紙芝居（動画ファイルは使わない：容量ゼロ・端末非依存・スキップ即応）。

| 幕 | 画 | 音 | 秒 |
|---|---|---|---|
| 1 掲示 | 黒に白抜き「令和八年十月十七日　二時間目／社会（歴史）中間試験」 | チャイム | 3.2 |
| 2 答案 | 机上の答案用紙（大問・設問・空欄）、壁時計 | 秒針 | 独白1行×n |
| 3 独白 | pack.prologue の各行＝彼女の心の声（「……全然、わかんない。」） | 秒針 | — |
| 4 まぶた | 上下から黒がまばたき。回を追うごとに長く、画がぼける | 秒針が遅く | 4.5 |
| 5 寝落ち | 全黒。机に伏す音。静寂 | こつん | 1.6 |
| 6 時渡り | 金泥の光が立ちのぼり、答案の**年号の数字が宙へ散る**。縦書き題字「夢のなか——平安」 | — | 4.5 |

- 「とばす」ボタン常設。おわったら既存の Guide（あそびかた）へ。
- 篇ごとの差分＝答案の中身（examTitle・設問）と独白（pack.prologue）。鎌倉は「大問二」の答案で再入眠。

### B. 目覚めアニメーション `WakeUp`（mode: epilogue を差し替え）
金の光→まぶたが逆にひらく→答案。**witnessed な設問のこたえが順に手書き風に現れる**。
独白（pack.epilogue、クリック送り）→締めの札（平安「つぎの頁——大問二『鎌倉時代』」／鎌倉「試験、終了。」）→題字へ。

### C. 試験データと充足判定
```ts
export interface ExamQ { id: string; eventId: string; q: string; a: string }
// Pack に examTitle: string / exam: ExamQ[] を追加
```
- **充足＝witnessed(eventId)**：learnedEvents に在る、または出来事の factId の栞を得ている（絵日記の既存判定と同一）。
- セーブ形式は**不変**（派生値なので保存しない）。旧セーブそのまま動く。
- 出題は各篇8問。全名所を巡り、暮らし（会話・栞）も拾えば全問埋まる配分：
  - 平安：桓武天皇／最澄／平等院鳳凰堂／院政／平氏／遣唐使／摂関政治／紫式部
  - 鎌倉：守護／征夷大将軍／執権／承久の乱／御成敗式目／北条時宗／運慶・快慶／後醍醐天皇

### D. 草子（絵日記帳）の再編＝参考資料化
- タブ構成：**「試験」・「年表」**・図譜・頁・栞（試験を先頭・既定に）。
- **試験タブ**＝和紙の帳面に**挟み込まれた現代のプリント**（白い紙・ゴシック・少し傾く）。
  夢のなかに持ち込んだ答案の写し、という舞台裏。世界観の対比をここでも使う。
  埋まった問いは走り書きのこたえ＋年・出来事名。未回答は「ゆかりの名所」の誘い。
- **年表タブ**＝全文公開の参考資料＋冒頭に時代のながれ帯（縄文〜明治、現在篇に印）。
  見た出来事に朱の「見た」印。未見は誘い文（既存 lockedHint を転用）。

### E. 音 `engine/school.ts`
チャイム（正打点の四音×2句）・秒針（1Hzの短いクリック）・机に伏す音（低いサイン減衰）。
すべて OscillatorNode 合成、try/catch で無音環境に無害。ユーザー操作（はじめから押下）後に鳴るので autoplay 制限に抵触しない。

### F. 文言の総点検（フレーム差し替え）
- prologue／epilogue（両篇）＝独白として全面書き直し。
- ホーム：副題「テストのつづきは、夢のなかで」、未刊の巻「次の試験」、脚注「試験範囲は、これからも増えてゆく」。
- 扉：「つづきから」→**「夢のつづきから」**。
- 案内（guideNote）：「名所にふれると頁がひらき、**答案のこたえが埋まってゆく**」を追記。

## 4. 北極星との整合確認

1. 読ませない → アニメは一行ずつ・自動送り。試験タブは任意閲覧。
2. 体験→気づき→さりげない答え合わせ → 「体験すると答案が埋まる」はこの原則そのもの。
3. ゲージを見せない → 点数・正誤判定なし。「埋まる／まだ」だけ。
4. 誰でも入れて深くも学べる → 年表全文公開で深掘りはいつでも。埋める遊びは体験必須。
5. もののあはれ → 夢from現実の枠は「覚めれば消える世界」の切なさを足す。終幕の独白で回収。
